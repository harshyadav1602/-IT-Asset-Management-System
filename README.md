# 💻 IT Asset Management System

A modern web-based **IT Asset Management System** developed using **Python (Flask)** and **PostgreSQL** to help organizations efficiently manage IT assets, employees, departments, maintenance records, complaints, and user authentication.

This project was developed as a major project to demonstrate practical full-stack development skills, database management, dashboard visualization, and secure authentication.

---

## 📌 Features

* 🔐 Secure User Authentication

  * User Registration
  * Login System
  * Forgot Password
  * Email OTP Verification
  * Password Reset

* 👨‍💼 Employee Management

  * Add Employee
  * Edit Employee
  * Delete Employee
  * Search Employee

* 💻 Asset Management

  * Add Assets
  * Update Assets
  * Delete Assets
  * Asset Allocation

* 🏢 Department Management

  * Manage Departments
  * Assign Employees

* 🛠 Maintenance Management

  * Create Maintenance Requests
  * Update Status
  * View Maintenance History

* 📢 Complaint Management

  * Register Complaints
  * Track Complaint Status
  * Resolve Complaints

* 📊 Dashboard

  * Total Employees
  * Total Assets
  * Total Departments
  * Total Complaints
  * Total Maintenance Requests
  * Charts using Chart.js

* 🌙 Modern User Interface

  * Responsive Design
  * Light/Dark Mode
  * Interactive Dashboard

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/harshyadav1602/-IT-Asset-Management-System.git
```

### Go to Project Folder

```bash
cd IT-Asset-Management-System
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure PostgreSQL

* Create a PostgreSQL database.
* Update your database credentials in the configuration file.

### Run the Application

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
